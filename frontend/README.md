# Campaign Manager Frontend

Frontend workspace implementing enablers `EN-05` to `EN-08`.

## Stack Decisions (SP-01)

- Build tool: `Vite`
- Server-state: `TanStack Query`
- Client-state: `Zustand`
- Session handling: in-memory JWT token

## Commands

- `yarn workspace campaign-manager-frontend dev`
- `yarn workspace campaign-manager-frontend build`
